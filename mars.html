Images and Info about NASA's Mar's Rovers

Click the button at right to choose a rover.

A background screen will appear at the left.

Push the background button for more info on that rover.

When ready enter the date of the pictures you would like to see.

The dates are available by sol day (martian day), or by earth date.

If pictures don't show it might be because there were no

pictures available from that date so choose another date.

Each day has multiple pictures so also enter a page number.

Default page number is 1 so it is not necessary to enter a page.

If pictures don't show, try entering a lower page value.
